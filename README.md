# IoT Based Predictive Maintenance System

## Description
This project monitors motor health using ESP32 and sensors.

## Features
- Real-time monitoring
- Fault detection
- Cloud upload (ThingSpeak)
- LCD display
- SD card logging

## Components
- ESP32
- ACS712
- ZMPT101
- ADXL345
- DHT11
- Relay
- LCD
- SD Card

## Working
Sensors → ESP32 → LCD + Cloud → Relay control

## Fault Conditions
- Temp > 50°C
- Current > 10A
- Vibration high
